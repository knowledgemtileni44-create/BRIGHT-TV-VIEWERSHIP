## BrightTV Viewership Analytics
This project analyzes user behavior and content consumption on BrightTV to generate insights that support Customer Value Management (CVM), improve engagement, and drive subscription growth.

## Project Overview
Using user profile and session-level viewership data, this project uncovers patterns in how users interact with the platform and identifies opportunities to optimize content strategy and retention.

## Objectives
Understand user behavior and viewing trends
Identify drivers of engagement
Detect peak and low consumption periods
Recommend strategies to grow and retain users

## Tools & Technologies
SQL – Data extraction and transformation
Databricks – Data processing
Excel – Data analysis
Power BI – Dashboard and visualization

## Key Insights
Engagement varies by time, day, and user segments
Clear peak and low viewing periods
Content preferences influence watch time and retention
## Recommendations
Optimize content scheduling for low-traffic periods
Use personalized recommendations
Align marketing with peak engagement times

## Deliverables
SQL queries and cleaned dataset
Excel analysis
Power BI dashboard
Final presentation with insights

## Conclusion
This project shows how data tools like SQL, Databricks, Excel, and Power BI can turn raw data into actionable insights for better decision-making and business growth.
